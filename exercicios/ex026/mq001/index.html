Estou aprendendo a criar Media Queries

As midia queries são utilizadas para criar layouts responsivos, ou seja, layouts que se adaptam a diferentes tamanhos de tela. Elas permitem que você defina estilos específicos para diferentes dispositivos, como smartphones, tablets e desktops.

Com as midia queries, você pode alterar a aparência do seu site com base na largura da tela, altura, resolução e outros fatores. Isso é especialmente útil para garantir que seu site seja acessível e fácil de usar em qualquer dispositivo.
